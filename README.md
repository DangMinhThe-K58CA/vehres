* Recommend: Using VueJs.
User:
- Login
- Manual register.
- Register with social networks: Facebook, Twiter, G+ v.v.v.
- Use Google maps:
	+ Get the nearest garages list by user’s current location.

	+ See detailed information of specify garage.

- Read news:
	+ Bookmark favorite news.
- Comment, rate garage.
